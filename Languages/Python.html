Python Beginner Questions & Answers

What is Python?
Python is a popular, high-level, interpreted programming language known for its readability and simplicity. It is widely used for web development, data analysis, automation, artificial intelligence, and more.
How do you print something in Python?
You can use the print() function. Example: print("Hello, world!")
How do you create a variable in Python?
Simply assign a value to a name. Example: x = 5
How do you write a comment in Python?
Use the # symbol. Example: # This is a comment
What is a list in Python?
A list is a collection of items that is ordered and changeable. Example: my_list = [1, 2, 3]
How do you write a loop in Python?
You can use a for loop. Example: for i in range(5): print(i)
How do you define a function in Python?
Use the def keyword. Example:
def greet(name):
print(f"Hello, {name}!")
What is indentation in Python and why is it important?
Indentation in Python refers to the spaces at the beginning of a code line. Python uses indentation to define blocks of code. It is required and enforces code structure, unlike many other languages that use braces {} or keywords. Incorrect indentation will result in an error.
How do you write an if statement in Python?
Use the if keyword. Example:
if x > 0:
print("x is positive")
How do you import a module in Python?
Use the import statement. Example: import math
What is a dictionary in Python?
A dictionary is a collection of key-value pairs. Example: my_dict = {'a': 1, 'b': 2}
How do you handle errors in Python?
Use try and except blocks. Example:
try:
result = 10 / 0
except ZeroDivisionError:
print("Cannot divide by zero")
How do you get user input in Python?
Use the input() function. Example: name = input("Enter your name: ")
