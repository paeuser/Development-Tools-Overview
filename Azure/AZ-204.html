Azure AI Quiz Questions

1. Which Azure resource provides language and vision services from a single endpoint?

  • Azure AI Language
  • Azure AI Vision
  • Azure AI Services
Explanation: Azure AI Services is a unified resource that offers multiple AI capabilities—like language, vision, speech, and decision—all accessible through a single endpoint. It simplifies integration and billing when using multiple services together.

2. You plan to create a simple chat app that uses a generative AI model. What kind of project should you create?

  • Azure AI Foundry project
  • Azure AI hub based project
  • Azure AI Custom Vision project
Explanation: Azure AI Foundry projects are designed for generative AI workloads like chatbots. They provide tooling and infrastructure specifically tailored for working with LLMs such as OpenAI models.

3. Which SDK enables you to connect to resources in a project?

  • Azure AI Services SDK
  • Semantic Kernel SDK
  • Azure AI Foundry SDK
Explanation: The Azure AI Foundry SDK is used to programmatically connect to and interact with resources in an Azure AI Foundry project. It includes tools to manage workflows, access models, and deploy AI applications.
