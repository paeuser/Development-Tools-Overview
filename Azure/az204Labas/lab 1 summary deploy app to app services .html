  1. Build and publish the app:
    dotnet publish -c Release -o publish
  2. Compress the publish folder into a zip file:
    Compress-Archive -Path .\publish\* -DestinationPath .\api.zip -Force
  3. Log in to Azure CLI and select the correct subscription:
    az login --tenant 7a16bfae-5ad8-4333-ae59-bdd3df252e95
    (Select your subscription when prompted)
  4. Set the subscription (if needed):
    az account set --subscription f6ab9759-ffe4-4b77-81f6-e55987b84565
  5. Deploy the zip file to Azure App Service:
    az webapp deploy --resource-group ManagedPlatformRG --name imgapieric --src-path .\api.zip --type zip

After these steps, your app will be live at https://imgapieric.azurewebsites.net/.
