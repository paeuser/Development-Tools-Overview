AZ-204 Labs Overview

These labs help you learn core Azure developer skills. Each lab builds practical experience with real Azure services.
Lab 01: Build a Web Application on Azure PaaS
Purpose: Learn how to create and deploy a web app using Azure App Service (PaaS).
  • Sign in to the provided Windows 11 VM.
  • Use tools like Edge, VS Code, and Terminal.
  • Create a web app in Azure.
  • Deploy your web app files using Kudu zip deployment.
  • Test your deployed app.
Lab 02: Implement Task Processing Logic with Azure Functions
Purpose: Understand how to use Azure Functions for serverless computing.
  • Create a simple Azure Function that echoes text sent via HTTP POST.
  • Learn how to trigger functions over HTTP.
Lab 03: Retrieve Azure Storage Resources with .NET SDK
Purpose: Learn to access Azure Storage (like blobs) from a .NET app.
  • Use the Azure Storage SDK in C# to connect to storage containers.
  • Retrieve metadata and URIs for blobs.
  • Practice accessing and displaying storage data.
Lab 04: Construct a Polyglot Data Solution (Cosmos DB)
Purpose: Work with Azure Cosmos DB (a NoSQL database) and integrate it with .NET.
  • Create Cosmos DB and a storage account.
  • Use C# to upload and access data in Cosmos DB.
  • Build a simple web interface to display Cosmos DB data.
Lab 05: Deploy Compute Workloads with Images and Containers
Purpose: Learn about containers and how to deploy them in Azure.
  • Use Docker to containerize a .NET app.
  • Push the container image to Azure Container Registry.
  • Deploy the container to Azure Container Apps.
How to Complete the Labs
  • Access the Lab Environment: Use the provided VM credentials (Admin / Pa55w.rd) and follow your instructor’s setup instructions.
  • Follow Each Lab’s Steps: Open the corresponding markdown file (e.g., AZ-204_lab_01.md) for detailed, step-by-step instructions.
  • Use the Right Tools: Most labs use Visual Studio Code, Terminal, and the Azure Portal.
  • Adapt to UI Changes: Azure’s interface may change. If you see differences, adapt and continue.
  • Test Your Work: Each lab ends with testing or validating what you built or deployed.

How to Get Started
  • To begin the AZ-204 labs, open the first lab instruction file:
    AZ-204/Instructions/Labs/AZ-204_lab_01.md
  • Each lab has its own markdown file in the same folder (e.g., AZ-204_lab_02.md, AZ-204_lab_03.md, etc.).
  • Open these files in order to follow the labs step by step.
  • You can use Visual Studio Code or any markdown viewer to read the instructions.
Tip: Start with AZ-204_lab_01.md and progress through the labs in order for the best learning experience.
