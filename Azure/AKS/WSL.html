WSL (Windows Subsystem for Linux)

What is WSL?

WSL (Windows Subsystem for Linux) lets you run a Linux environment directly on Windows without needing a virtual machine. You can run Linux command-line tools, utilities, and apps natively alongside your Windows apps. It’s great for developers who want Linux tools but work on Windows.

  • Official docs: https://learn.microsoft.com/windows/wsl/

How to Open WSL (Ubuntu) Again After Closing

  • From Start Menu:
    • Click Start (Windows icon).
    • Type Ubuntu.
    • Click the Ubuntu app to open your WSL terminal.

What is jq?

jq is a lightweight and flexible command-line tool for processing JSON data.

  • On Windows: If you’re using WSL (Linux environment on Windows), you can install jq inside WSL with:
    sudo apt update
    sudo apt install jq

How to Install zsh (Z Shell) in WSL/Ubuntu

Summary:
sudo apt install zsh is a command used on Debian-based Linux systems (like Ubuntu) to install the Z Shell (zsh), a popular alternative to the default shell. The sudo part runs the command with administrator (root) privileges needed for installation. After installing, you can switch to zsh for a more powerful and customizable command-line experience.

How to install zsh:
  1. Open your terminal.
  2. Run sudo apt update to refresh package lists.
  3. Run sudo apt install zsh to install zsh.
  4. After installation, start zsh by typing zsh in the terminal.
