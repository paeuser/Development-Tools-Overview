What is a Jupyter Notebook?

A Jupyter Notebook is an interactive document where you can write and run code, add notes, and see results all in one place. It is great for learning, experimenting, and sharing code, especially in Python.

Simple Tutorial: How to Use a Jupyter Notebook

  1. Open a Notebook: Double-click a .ipynb file or open it in VS Code or GitHub Codespaces.
  2. Cells: Notebooks are made of cells. Each cell can be code (Python) or text (Markdown).
  3. Run Code: Click a code cell and press Shift+Enter to run it. The output appears below the cell.
  4. Add Notes: Use markdown cells to write explanations, titles, or instructions.
  5. Edit and Experiment: Change code or text, then run again to see new results.
  6. Save: Your work is saved automatically, but you can also use Ctrl+S.
# Example: Simple Python code in a notebook
print("Hello, world!")

Tips for Beginners

  • Use code cells for Python code.
  • Use markdown cells for notes and explanations.
  • You can add, delete, or move cells as you like.
  • Notebooks are perfect for step-by-step learning and trying out ideas.

How to Create a Jupyter Notebook Programmatically

  1. Import Required Libraries: Use import nbformat to work with notebook files.
  2. Create a Notebook Object: Use nb = nbformat.v4.new_notebook() to start a new notebook.
  3. Add Cells: Add markdown and code cells using nbformat.v4.new_markdown_cell() and nbformat.v4.new_code_cell().
  4. Save the Notebook: Write the notebook to a file with nbformat.write(nb, file).

This lets you generate .ipynb files with Python code, which is useful for automation or sharing templates.

Before You Start: Install the Jupyter and Python Extensions

  • To use Jupyter Notebooks in VS Code, you need to install the Jupyter extension from the Extensions Marketplace.
  • You also need the Python extension to run Python code in notebooks.
  • Search for "Jupyter" and "Python" in the Extensions view (Ctrl+Shift+X) and click Install for both.
  • These extensions let you open, edit, and run .ipynb files directly in VS Code.

Getting Started with Jupyter Notebooks in VS Code

  • Watch this video: Getting Started with Jupyter Notebooks in VS Code
  1. Go to the Command Palette: Press Ctrl+Shift+P and type Create: New Jupyter Notebook to quickly create a new notebook file in VS Code.
  2. Before you start writing code, connect to the kernel: The kernel is the computational engine that runs your code. Make sure the kernel is connected (look for the status in the top right of the notebook) before running any code cells.

What is the .venv Folder and Why Was it Created?

  • When you use the Command Palette in VS Code to create or open a Jupyter Notebook, VS Code may prompt you to select or create a Python environment for your project.
  • If you choose to create a new environment, VS Code will often create a .venv folder in your project directory. This stands for "virtual environment."
  • A virtual environment is an isolated space for Python, allowing you to install packages and dependencies just for your project, without affecting other projects or your system Python.
  • This helps keep your project organized and avoids version conflicts between different Python projects.
  • You can see and manage the .venv folder in your workspace. It is safe to keep it in your project folder.

Working with Cells in Jupyter Notebooks

  • You can add cells to your notebook. Cells are where you write and execute code.
  • You can also add markdown cells for notes, explanations, and formatting.
  • Cells can be deleted, moved, or edited as needed to organize your notebook.

  • plt.xlabel('Price'): Labels the x-axis as "Price".
  • plt.ylabel('Stock'): Labels the y-axis as "Stock".
  • plt.title('Product Price vs Stock'): Adds a title to the plot.
  • plt.show(): Displays the plot in the notebook.

This is a simple way to visualize the relationship between product price and stock in your data.
