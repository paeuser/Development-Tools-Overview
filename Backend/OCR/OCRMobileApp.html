Project Overview

  • Built with Vue.js, using a modular component structure (see src/components/).
  • Main entry: src/main.js; root component: src/App.vue.
  • Uses static assets and JS libraries (e.g., public/libs/js/web-capture.js) for image/document capture.
  • Bundled and served with Vite (vite.config.js).
  • Designed for both desktop and mobile browsers, with responsive layout and mobile web app meta tags.

How the App Works

  1. The user interacts with the UI rendered by Vue components (App.vue and components/).
  2. The app can capture images/documents using the web-capture.js library.
  3. Captured images are processed for OCR, and results are displayed in the app interface.
  4. The app structure supports easy extension and backend integration.

Project Files

  • src/App.vue: Main Vue component, contains the app's core logic and layout.
  • src/main.js: Entry point, mounts the Vue app to #app in this HTML.
  • src/components/: Reusable Vue components for UI and features.
  • public/libs/js/web-capture.js: JS library for image/document capture.
  • vite.config.js: Vite configuration for development/build.
  • README.md: Project documentation and usage instructions.

For more details, see the README.md and the source files in the src/ directory.
