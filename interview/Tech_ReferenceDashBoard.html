YARP (Yet Another Reverse Proxy).
It acts as a gateway for all OCR-related API requests,
providing authentication, routing, rate limiting, and
error handling for downstream services such as Assurity.ApiWrapper.Service.

Rate Limiting
Implements IP-based rate limiting using AspNetCoreRateLimit to prevent abuse.
Limits are configurable in appsettings.json (e.g., 5 requests per 10 seconds).
API key must be provided in the api-Key header
