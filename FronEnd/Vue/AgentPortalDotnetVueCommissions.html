The Commissions module provides agents and business users with a comprehensive view of their commission earnings, debt balances, and related financial details. It allows users to:
  • View commission summaries, year-to-date and year-over-year comparisons.
  • Access detailed breakdowns by policy and by writing agent.
  • Download commission and debt data for further analysis.
  • See important dates for commission processing and statement availability.
  • Review direct deposit information and other related resources.

This module is designed to help agents understand their earnings, track outstanding debts, and access all relevant financial information in one place, supporting better business decisions and transparency.

How this UI works (for technical users):
  • Vue.js & Vuetify: The UI is built with Vue.js and Vuetify for a modern, responsive experience that adapts to both desktop and mobile devices.
  • Component-based: The interface is split into reusable components (e.g., AgentStatement, DirectDeposit, ExpandedDetails, ExpandedDetailsTable) for maintainability and modularity.
  • Views & Tabs: Main screens (like HomeView, DebtView) aggregate these components, presenting summary cards, tables, and filters. Tabs allow users to switch between commission and debt breakdowns (by policy, by agent, secured/unsecured debt).
  • Data Tables: Vuetify's data tables display large sets of commission and debt records, with sorting, filtering, and expandable rows for more detail. These tables are responsive and adapt to screen size.
  • State Management: Pinia (Vue's store) manages data such as commission details, debt, filters, and loading states, ensuring a reactive and consistent UI.
  • Download & Export: Users can export data (e.g., to XLSX) for offline analysis.
  • Responsive Design: The UI adapts for desktop and mobile, showing different layouts and controls as appropriate. Mobile layouts use stacked cards and collapsible sections for readability.
  • Custom Styling: Styles are managed with PostCSS (PCSS) for consistent branding and layout.

This architecture ensures the Commissions module is user-friendly, scalable, and easy to maintain, while providing all the features agents and business users need to manage their commission-related activities.
