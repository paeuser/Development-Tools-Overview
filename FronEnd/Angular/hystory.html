Angular Version History

This page outlines major Angular versions, their release dates, and key new features introduced in each version.

Version Release Date Key New Features
AngularJS (1.x) October 2010 Original framework, two-way data binding, dependency injection, directives, MVC architecture.
Angular 2 September 2016 Complete rewrite, TypeScript support, components-based architecture, improved dependency injection, mobile support.
Angular 4 March 2017 Smaller and faster apps, improved *ngIf and *ngFor, Angular Universal (server-side rendering), TypeScript 2.1/2.2 support.
Angular 5 November 2017 Build optimizer, improved compiler, HttpClient, internationalized number/date/currency pipes.
Angular 6 May 2018 ng update, ng add, Angular Elements, tree-shakable providers, RxJS 6 support.
Angular 7 October 2018 CLI prompts, virtual scrolling, drag & drop, improved accessibility, Angular Material & CDK updates.
Angular 8 May 2019 Differential loading, dynamic imports for lazy routes, web workers, builder APIs.
Angular 9 February 2020 Default Ivy compiler, smaller bundle sizes, improved type checking, faster testing.
Angular 10 June 2020 New date range picker, stricter project setup, TypeScript 3.9 support, optional stricter settings.
Angular 11 November 2020 Faster builds, updated Hot Module Replacement (HMR), improved logging, stricter types, component test harnesses.
Angular 12 May 2021 Nullish coalescing, strict mode by default, Webpack 5 support, improved styling, deprecated View Engine.
Angular 13 November 2021 Removal of View Engine, persistent build cache, dynamic component creation, improved Angular Package Format (APF).
Angular 14 June 2022 Standalone components, typed forms, extended template diagnostics, streamlined CLI, improved accessibility.
Angular 15 November 2022 Directive composition API, image directive, improved standalone APIs, better SSR support.
Angular 16 May 2023 Signals for reactivity, server-side hydration, esbuild support, improved developer experience.
Angular 17 November 2023 Deferrable views, built-in control flow, SSR improvements, new documentation site, performance boosts.

For the latest Angular features and release notes, visit the official Angular releases page.
