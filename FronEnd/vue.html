Composition API in Vue 3 The Composition API is a new way to organize and reuse logic in Vue components, offering better flexibility and scalability.
Components Components are reusable instances with their own structure, logic, and style, used to build applications in Vue.js.
Computed properties vs methods Computed properties are cached based on their dependencies, while methods are re-executed every time they are called.
Directives Directives are special tokens in the markup that tell the library to do something to a DOM element, such as v-if, v-for, and v-bind.
Filters Filters are used to format or transform data in the template, such as formatting dates or numbers.
Fragments in Vue 3 Fragments allow components to return multiple root nodes, eliminating the need for a single root element.
Instance The Vue instance is the root of a Vue application, managing the data, methods, and lifecycle hooks.
Key attribute The "key" attribute is used to identify VNodes and optimize rendering by reusing elements efficiently.
Lifecycle The lifecycle includes stages like creation, mounting, updating, and destruction, with hooks for each stage.
Mixins Mixins are a way to distribute reusable functionalities for Vue components.
New in Vue 3 Vue 3 introduces the Composition API, improved performance, better TypeScript support, and a smaller bundle size compared to Vue 2.
Router Vue Router is the official router for Vue.js, enabling navigation between pages and components in a Vue application.
Scoped styles Scoped styles ensure that CSS rules apply only to the component they are defined in, preventing conflicts with other components.
Suspense feature in Vue 3 Suspense is a feature that allows you to handle asynchronous dependencies in components, showing fallback content while waiting for data to load.
Teleport feature in Vue 3 Teleport allows you to render a component's template in a different part of the DOM outside its parent component.
Virtual DOM The virtual DOM is a lightweight representation of the real DOM, used to optimize rendering by updating only the necessary parts of the DOM.
Vue CLI Vue CLI is a command-line tool for scaffolding and managing Vue.js projects with features like plugins and configurations.
Vuex Vuex is a state management library for Vue.js applications, providing a centralized store for all components.
v-show vs v-if v-show toggles the visibility of an element using CSS, while v-if adds or removes the element from the DOM.
Async components Async components allow you to load components lazily, improving performance by splitting the application into smaller chunks.
Custom directives Custom directives are user-defined directives that extend Vue's functionality by adding custom behavior to DOM elements.
Dynamic components Dynamic components allow you to switch between components dynamically using the <component> element and the is attribute.
Event handling Event handling in Vue is done using the v-on directive or the @ shorthand to listen to DOM events and execute methods.
Reactivity system The reactivity system in Vue tracks changes to data and automatically updates the DOM to reflect those changes.
Slots Slots are placeholders in a component's template that allow you to pass content from a parent component to a child component.
Transitions and animations Transitions and animations in Vue are used to apply effects when elements are added, updated, or removed from the DOM.
