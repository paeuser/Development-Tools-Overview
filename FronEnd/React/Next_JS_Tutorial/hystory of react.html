History of React

When was React invented?

React was invented by Jordan Walke, a software engineer at Facebook, and was first released in May 2013.

React Versions by Year

  • 2013: React 0.3.0 (Initial public release)
  • 2014: React 0.9.0, 0.10.0, 0.11.0, 0.12.0
  • 2015: React 0.13.0, 0.14.0, 0.15.0
  • 2016: React 15.0.0
  • 2017: React 16.0.0
  • 2018: React 16.3, 16.4, 16.5, 16.6, 16.7
  • 2019: React 16.8 (Hooks), 16.9, 16.10, 16.11, 16.12, 16.13
  • 2020: React 17.0.0
  • 2022: React 18.0.0
  • 2023: React 18.x updates
  • 2024: React 18.x updates
  • 2025: Latest React 18.x (as of June 2025)
