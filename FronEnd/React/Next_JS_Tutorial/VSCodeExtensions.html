ES7 React/Redux/GraphQL/React-Native VSCode Extension

The ES7 React/Redux/GraphQL/React-Native snippets extension for VSCode provides JavaScript and React/Redux snippets to speed up your workflow.

  • Quickly generate boilerplate code for React components
  • Supports Redux, GraphQL, and React Native
  • Improves productivity with handy shortcuts

How to Install

  1. Open VSCode
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for ES7 React/Redux/GraphQL/React-Native snippets
  4. Click Install

Example Snippet

rafce
// Generates a React Arrow Function Component with export

Tailwind CSS IntelliSense
