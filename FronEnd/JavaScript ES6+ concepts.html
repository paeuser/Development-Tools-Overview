JavaScript ES6+ Concepts

Key Features Introduced in ES6 and Beyond

  • let & const: Block-scoped variable declarations.
  • Arrow Functions: Shorter function syntax: (a, b) => a + b
  • Template Literals: String interpolation with backticks: `Hello, ${name}!`
  • Destructuring: Extract values from arrays/objects: const [a, b] = arr;
  • Default Parameters: Set default values for function parameters.
  • Rest & Spread Operators: ... for variable arguments and copying/merging arrays/objects.
  • Classes: Syntactic sugar for constructor functions and inheritance.
  • Modules: import and export for modular code.
  • Promises & async/await: Modern asynchronous programming.
  • Map & Set: New collection types for unique values and key-value pairs.
  • Optional Chaining & Nullish Coalescing (ES2020): obj?.prop, value ?? default

Example: Arrow Function & Destructuring

const add = (a, b) => a + b;
const person = { name: "Alice", age: 30 };
const {{ name, age }} = person;

Example: Async/Await

async function fetchData() {
  const response = await fetch('https://api.example.com/data');
  const data = await response.json();
  return data;
}

Why Learn ES6+?

  • Cleaner, more readable code
  • Modern syntax is widely used in frameworks (React, Vue, Angular)
  • Better support for modular, maintainable applications
